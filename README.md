## ✨ Features

### 👤 User Features
- ✅ **User Authentication** - Register, login, and logout functionality
- ✅ **Password Recovery** - Reset password with 6-digit verification code
- ✅ **Product Browsing** - View products with categories and filters
- ✅ **Live Search** - Real-time product search with dropdown results
- ✅ **Shopping Cart** - Add/remove products, update quantities
- ✅ **Order Management** - Place orders and track order status
- ✅ **Order Tracking** - Track orders using unique tracking number
- ✅ **Product Reviews** - Rate and review products (star rating system)
- ✅ **User Profile** - View order history and account details

### 🛍️ Product Features
- ✅ **Multi-image Support** - Upload up to 4 images per product (separated by `|`)
- ✅ **Discount System** - Percentage-based discounts on products
- ✅ **Category Filtering** - Filter products by categories
- ✅ **Stock Management** - Track product inventory
- ✅ **Product Features** - Add detailed specifications for each product

### 🔧 Admin Panel Features
- ✅ **Dashboard** - View statistics (products, users, orders, cart items)
- ✅ **Product Management** - Add, edit, delete products
- ✅ **Category Management** - Add, edit, delete categories
- ✅ **Order Management** - Update order status (pending → paid → shipped → delivered)
- ✅ **Comment Moderation** - Approve or reject user comments

### 💳 Checkout Process
1. **Shopping Cart** - Review items and proceed to checkout
2. **Customer Information** - Enter shipping details (name, phone, address)
3. **Payment Gateway** - Simulated payment page
4. **Order Confirmation** - Receive tracking number and order confirmation
